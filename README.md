# 🧠 Home Lab Network Setup (Cisco Packet Tracer)

## 📘 Project Overview
This is a complete home lab network simulation using Cisco Packet Tracer. It includes VLAN segmentation, OSPF routing, DHCP setup.

## 🖼️ Topology Diagram
![Network Topology](topology.png)

## 🧱 Key Features
- VLANs with trunking
- Inter-VLAN routing via Router-on-a-Stick
- OSPF dynamic routing
- Device configs included

## 📁 Folder Structure
- `config/` → All router and switch configuration files.
- `pkt-files/` → Packet Tracer `.pkt` file.
- `docs/` → Detailed documentation and planning notes.

## 🛠️ Tools Used
- Cisco Packet Tracer
- Markdown for documentation

## 🚀 How to Use
1. Download the `.pkt` file from `pkt-files/`.
2. Open in Cisco Packet Tracer.
3. Configure or test as needed.
4. Use the `config/` files as a reference or backup.
